📚 Правила логарифмов

1. Определение: logₐ b = c ⇔ aᶜ = b
2. Сложение: logₐ x + logₐ y = logₐ(x·y)
3. Вычитание: logₐ x - logₐ y = logₐ(x/y)
4. Степень: logₐ(xᵏ) = k · logₐ x
5. Переход: logₐ b = 1 / logᵦ a

Работа с инструментами:
• Основание: x = b(1/c)
• Аргумент: x = ac
• Значение: x = logₐ b

ℹ️ О проекте

«Помощь в решении логарифмов» — это некоммерческий образовательный проект, созданный в 2026 году.

Разработкой проекта занимался учащийся МБОУ «Ермаковская СШ №1 имени В. П. Брагина» Гордеев Никита Александрович под руководством учителя математики Исаковой Варвары Владимировны.

Главная цель проекта — упростить освоение темы логарифмов и предоставить школьникам доступный инструмент для самопроверки.

Содержание учебных материалов потребуется для сдачи ВПР для 10-х классов, а также для ЕГЭ.
